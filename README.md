### Number Row Characters

* Shift-1: \!
  - Exclamation mark

* Shift-2: \@
  - At symbol

* Shift-3: \#
  - Hash

* Shift-4: \$
  - Dollar sign

* Shift-5: \%
  - Percent sign

* Shift-6: \^
  - Caret

* Shift-7: \&
  - And

* Shift-8: \*
  - Asterisk

* Shift-9: \(
  - Left parenthesis

* Shift-0: \)
  - Right parenthesis

* Shift--: \_
  - Underscore

* Shift-=: \+
  - Plus sign
