### Number Row Characters

1\. **Shift-1: \!**
  - Exclamation mark

2\. **Shift-2: \@**
  - At sign

3\. **Shift-3: \#**
  - Hash

4\. **Shift-4: \$**
  - Dollar 

5\. **Shift-5: \%**
  - Percent 

6\. **Shift-6: \^**
  - Hat

7\. **Shift-7: \&**
  - And

8\. **Shift-8: \***
  - Star

9\. **Shift-9: \(**
  - Left parenthesis

10\. **Shift-0: \)**
  - Right parenthesis

11\. **Shift--: \_**
  - Underline

12\. **Shift-=: \+**
  - Plus
